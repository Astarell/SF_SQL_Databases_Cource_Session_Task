# [Сессионное задание] Итоговый проект. База данных

## Структура
1. Решения задач по представленным БД находятся в соответствующих папках (напр. "db_1" содержит в себе скрипты для "База данных 1. Транспортные средства").
2. В каждой папке располагаются скрипты "DB{номер бд}_create_tables" и "DB{номер бд}_insert_test_data" для создания таблиц и связей между ними и вставки тестовых данных соответственно.
3. Решенные задачи согласно своему номеру представлены файлами "DB{номер бд}_task{номер задачи}"

## Как тестировать
1. Выполнение задач происходило в программе DBeaver (Community Edition), использовалась PostgreSQL. Скрипты отлично будут запускаться в той же среде.
